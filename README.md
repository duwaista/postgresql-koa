## Первый запуск проекта
1. Установка зависимостей:
``` npm install ```

2. Установка Knex:
``` npm i knex -g ```

3. Создать файл `knexfile.js` на основе `knexfile.template.js`
4. EZ

#### Запуск проекта:
``` npm start ```

#### Запуск миграции:
``` npm run migrate ```

#### Откат миграции:
``` npm run unmigrate ```

#### Запуск Seed:
``` npm run seed ```
